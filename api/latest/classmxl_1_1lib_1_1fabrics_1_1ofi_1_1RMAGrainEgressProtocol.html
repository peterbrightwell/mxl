void mxl::lib::fabrics::ofi::RMAGrainEgressProtocol::registerMemory ( std::shared_ptr< Domain >  domain)
finalvirtual

Register local memory regions used in this protocol.

Parameters
domainThe domain to register memory with.
Note
This is used to register additional local memory regions. One example of this, is in the case where there is a metadata header that needs to be transferred along with the actual data, and the header is stored in a different memory region from the data. And such memory region can only be created after the protocol template creates the protocol instance since the header content is only known after the instance is created.
This is a NOP for this protocol
