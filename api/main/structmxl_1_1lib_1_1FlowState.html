◆ syncCounter

std::uint32_t mxl::lib::FlowState::syncCounter

32 bit word used synchronization between a writer and multiple readers. This value can be used by futexes. When a FlowWriter commits some data (a grain, a slice, etc) it will increment this value and then wake all FlowReaders waiting on this memory address.
