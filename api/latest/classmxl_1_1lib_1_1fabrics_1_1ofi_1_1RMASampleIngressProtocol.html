Helper function to create an AudioBounceBuffer based on the given audio data layout and maximum synchronous batch size. Entries are as big as the maximum number of samples that can be transferred in a single batch, which is determined by maxSyncBatchSize. The number of entries is determined by how many batches are needed to cover the entire history buffer.
