Detailed Description

Monitors flows on disk for changes.

A flow writer is looking for changes to the {mxl_domain}/{flow_id}.mxl-flow/access file. This file is 'touched' by readers when they read a grain, which will trigger a 'FlowInfo.lastRead` update.
