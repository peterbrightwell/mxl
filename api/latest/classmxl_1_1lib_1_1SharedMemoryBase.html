protected

Upgrade the lock held by the process for the shared memory file that is mapped by this object to an exclusive lock.

Returns
Returns true if the upgrade was successful. Returns false if another process also holds a shared lock and acquiring an exclusive lock would be a blocking operation.
Exceptions
std::runtime_errorIf this is a read-only mapping of the memory or if the object is in an invalid state.
