◆ mxlCreateFlowWriter()

MXL_EXPORT mxlStatus mxlCreateFlowWriter ( mxlInstance  instance,
char const *  flowDef,
char const *  options,
mxlFlowWriter *  writer,
mxlFlowConfigInfo *  configInfo,
bool *  created 
)

Attempts to create a flow writer for a given flow definition. If the flow does not exist already, it is created and 'created' will be set to true. If the flow exists, it will be opened and 'created' will be set to false. If the flow exists already it is not guaranteed that the flow definition of the existing flow is exactly equal to the flow definition supplied in flowDef. The definition of the existing flow can be retrieved using mxlGetFlowDef().

IMPORTANT NOTE
The flow definition (flowDef) is expected to have well defined fields such as label, description and grouphint tag. This is a responsibility of the media function implementer. These fields should refer to the media function instance name. For example, on a large system with mutliple software defined decoders running simultaneously, each decoder instance should use a unique name in the grouphint tag to identify itself. Please read carefully the 'natural groups' section of the group hint tag specification.

Example fields for an hypothetical video decoder media function instance named "Decoder 1":

  • label: "Decoder 1 Video Output"
  • description: "Decoder 1 MXL Flow Video Output"
  • grouphint tag: "Decoder 1:Video"

Example fields for a multi-flow audio receiver media function instance named "Audio Receiver A": Flow 1:

  • label: "Audio Receiver A Audio Output #1"
  • description: "Audio Receiver A MXL Flow Audio Output #1"
  • grouphint tag: "Audio Receiver A:Audio #1"

Flow 2:

  • label: "Audio Receiver A Audio Output #2"
  • description: "Audio Receiver A MXL Flow Audio Output #2"
  • grouphint tag: "Audio Receiver A:Audio #2"

Having a well formed grouphint tag refering to the media function instance name is essential to ensure proper flow discovery and management by higher level applications using MXL.

Parameters
[in]instanceThe mxl instance created using mxlCreateInstance
[in]flowDefThe flow definition from which a flow should be created if there is not already a flow with the same flow id.
[in]options(optional) Additional options, can be NULL
[out]writerA pointer to a memory location where the created flow writer will be written.
[out]configInfo(optional) Used to return information about the opened flow. Can be NULL.
[out]created(optional) If not NULL, will be set to true if a new flow was created, false if an existing flow was opened.
