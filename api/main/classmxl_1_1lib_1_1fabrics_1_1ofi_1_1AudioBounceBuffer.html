Detailed Description

A bounce buffer for audio samples.

For audio samples, a bounce buffer is used to receive the samples from the network before copying them to the final destination region. This is necessary because the samples to transfer on the Initiator are scattered at multiple locations (one per channel), but only a single memory address can be provided for the destination.
