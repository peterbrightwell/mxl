std::filesystem::path mxl::tests::getDomainPath ( )
inline

Simple utility function to get the domain path for the tests. This will return a path to /dev/shm/mxl_domain on Linux and a path in the user's home directory ($HOME/mxl_domain) on macOS.

Returns
The path to the mxl domain directory. The directory may not exist yet.
