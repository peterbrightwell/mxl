Blocking wait function for a specific grain at a specific index. The index must be greater than or equal to the current tail index of the flow.

Parameters
in_indexThe grain index.
in_minValidSlicesThe expected number of valid slices in the returned mxlGrainInfo.
in_deadlineThe point in time of Clock::Realtime at which to stop waiting.
Returns
A status code describing the outcome of the call. Please note that this method will never return MXL_ERR_TIMEOUT, because the actual error that is being encountered in this case is MXL_ERR_OUT_OF_RANGE_TOO_EARLY, even after waiting.
Note
Please note that contrary to the various overloads of getGrain, this method does not update the flow access time.
