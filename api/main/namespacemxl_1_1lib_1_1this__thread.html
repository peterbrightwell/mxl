Sleep for the specified amount of time to pass on the specified clock.

Parameters
[in]durationthe amount of time to sleep for,
[in]clockthe clock the specified duration refers to.
Returns
The remaining time of the originally specified duration that was not spent sleeping because the sleep was interrupted for any reason.

If the remaining time is non-zero the condition that caused the sleep to abort can be obtained by inspecting errno.
