Detailed Description

Reliable Connected (RC) Target implementation.

An RC Target uses connection-oriented endpoint type. It listens for incoming connection requests from an Initiator to establish connection. Once connected, it polls its completion queues to see if new data can be consumed. be read from the connected endpoint.
