uint32_t mxlDiscreteFlowConfigInfo_t::grainCount

How many grains in the ring buffer. This should be identical to the number of entries in the {mxlDomain}/{flowId}/grains/ folder. Accessing the shared memory section for that specific grain should be predictable.

uint32_t mxlDiscreteFlowConfigInfo_t::sliceSizes[MXL_MAX_PLANES_PER_GRAIN]

Length of a slice in bytes. A slice refers to the elemental data type that can be written and comitted to a grain. For video, this is a line of a V210 picture including any padding. For data, this is just a single byte.
