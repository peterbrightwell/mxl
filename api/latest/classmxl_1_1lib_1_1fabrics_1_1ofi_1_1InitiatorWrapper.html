Detailed Description

A wrapper around Initiator implementations.

This wrapper creates an unspecified initiator that can be configured for a specific type by calling the setup() method.

void mxl::lib::fabrics::ofi::InitiatorWrapper::addTarget ( TargetInfo const &  targetInfo)

Add a target to the initiator.

The is a non-blocking operation. The local endpoint for this target will only actively start connecting to its remote counterpart when control is passed to makeProgress() or makeProgressBlocking(). The local endpoint for this target will only accept write requests after the progress functions return false.
