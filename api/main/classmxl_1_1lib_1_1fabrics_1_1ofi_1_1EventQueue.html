◆ readBlocking()

std::optional< Event > mxl::lib::fabrics::ofi::EventQueue::readBlocking ( std::chrono::steady_clock::duration  timeout)

Perform a blocking read on the event queue.

This function will block until the specified timeout has elapsed, the process is signalled, or an event is available on the queue. If no event was available after the timeout, the function returns std::nullopt. If the process is signalled while blocking on this function, an exception will be thrown.
